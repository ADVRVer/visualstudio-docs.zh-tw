# <a name="handling-expired-subscriptions"></a>處理過期的訂閱

當使用者的訂用帳戶到期時，在入口網站中會以紅色長條在訂閱者清單中的使用者名稱旁邊表示。 若要讓這些訂閱者可以繼續使用 Visual Studio 訂閱權益，您必須更新他們的授權。 

![訂閱逾期的清單](_img\expired-subscriptions\expired-list.png) 

若要更新訂閱，請連絡大量授權的轉銷商或 Microsoft 賣方。 將您的合約編號和購買帳戶號碼提供給他們。 您的轉銷商或 Microsoft 賣方將引導您完成更新，或進行購買其他訂閱的程序。 下好訂單或更新合約之後，您就可以使用新的訂閱，並將其指派給訂閱者。
