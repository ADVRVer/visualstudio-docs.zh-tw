# 其他 MSBuild 資源
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

如果您正在嘗試解決問題，而在文件中找不到需要的資訊，您可以諮詢其他資源，例如知識庫 \(Knowledge Base\) 文件、開發人員社群或產品支援。  
  
## 知識庫文件  
 您可以在線上知識庫 \(KB\) 搜尋有關產品問題的文章。  並非所有的問題都有對應的知識庫文章，不過有許多客戶都遇到過的錯誤通常會有文件記錄。  您可以進入 [知識庫 \(MSBuild 搜尋結果\)](http://search.microsoft.com/supportresults.aspx?q=knowledge+base+MSBuild&x=0&y=0) 在網站上的知識庫文件。  
  
## 社群論壇  
 在論壇中您可以和其他開發人員以及 Microsoft 員工互動。  如果您所遇到的錯誤找不到解決方式，您可以將該錯誤的問題張貼到論壇上。  您也可以搜尋新聞群組，查看是否有其他人已張貼相同的問題。  您可以進入在 [MSDN 論壇](http://go.microsoft.com/fwlink/?LinkId=48931) 網站上的Visual Studio MSBuild 的論壇。  
  
 您可以進入在 [Channel 9 MSBuild](http://channel9.msdn.com/) [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] 網站的社群資源。  
  
## 產品支援  
 如果您在嘗試其他資源後還是有問題，則可以造訪 [Microsoft 說明及支援](http://go.microsoft.com/fwlink/?linkid=1782) 網站，連絡 Microsoft 支援服務。  如需您所在地的產品支援資訊，請參閱[告訴我們](../ide/talk-to-us.md)。
