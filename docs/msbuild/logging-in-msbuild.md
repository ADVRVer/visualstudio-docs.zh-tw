# MSBuild 中的記錄
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

記錄提供讓您監視建置進度的方式。  記錄會將建置事件、訊息、警告和錯誤擷取到記錄檔中。

 描述記錄如何在多處理器的環境下運作，以及兩個多處理器記錄模型。

 概述如何建立能夠辨識多處理器的記錄器，以及如何使用 ConfigurableForwardingLogger。
