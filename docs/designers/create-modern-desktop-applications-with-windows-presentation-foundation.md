# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>使用 Windows Presentation Foundation 建立新式桌面應用程式

Visual Studio 中的 Windows Presentation Foundation (WPF) 為開發人員提供了統一的程式撰寫模型，可在 Windows 上建置新式的企業營運桌面應用程式。  

如需深入的逐步解說，請從 [WPF 使用者入門](../designers/getting-started-with-wpf.md)開始，並遵循其次要主題。

如需可用工具的比較，請參閱[在 Visual Studio 和 Blend for Visual Studio 中設計 XAML](../designers/designing-xaml-in-visual-studio.md)。
