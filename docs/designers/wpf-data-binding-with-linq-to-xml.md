# 使用 LINQ to XML 進行 WPF 資料繫結
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

本節提供的資訊關於如何使用 LINQ to XML 做為 Windows Presentation Foundation \(WPF\) 應用程式中資料繫結的資料來源。這個案例會依賴 <xref:System.Xml.Linq.XAttribute?displayProperty=fullName> 和 <xref:System.Xml.Linq.XElement?displayProperty=fullName> 的*「動態屬性」*。這些動態屬性也有在本節中加以說明。  
  
## 在本節中  
  
|主題|描述|  
|--------|--------|  
|[使用 LINQ to XML 進行 WPF 資料繫結概觀](../designers/wpf-data-binding-with-linq-to-xml-overview.md)|介紹 <xref:System.Xml.Linq> 命名空間所提供的動態資料繫結功能，並說明如何使用這些功能做為 WPF 中，使用者介面元件的資料來源。|  
|[LINQ to XML 動態屬性](../designers/linq-to-xml-dynamic-properties.md)|提供由 <xref:System.Xml.Linq.XAttribute> 和 <xref:System.Xml.Linq.XElement> 類別公開之動態屬性的參考資訊。|  
|[使用 LINQ to XML 進行 WPF 資料繫結範例](../designers/wpf-data-binding-using-linq-to-xml-example.md)|提供會將使用者介面元件繫結至 XML 資料來源的 WPF 範例。|
