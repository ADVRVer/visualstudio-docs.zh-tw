# <a name="choose-breakpoints-dialog-box"></a>[選擇中斷點] 對話方塊
**選擇中斷點**對話方塊隨即開啟，當您嘗試從模稜兩可的輸入中設定中斷點**新增中斷點** 對話方塊。 模稜兩可的常見範例是輸入的多載函式名稱。  
  
 **中斷點**  
 清單可能中斷點可以設定為基礎的模稜兩可的輸入。 您可以選取要設定的一或多個中斷點。  
  
 **All**  
 選取所有列中的中斷點**中斷點**方塊。  
  
 **無**  
 清除所有中斷點中所列**中斷點**方塊。
