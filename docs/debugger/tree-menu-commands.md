# 樹狀結構功能表命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[**樹狀結構**\] 功能表提供展開或摺疊檢視的命令。  
  
 **展開次層子目錄**  
 顯示系統檢視中目前選取項目的下一層，即可展開該項目。  
  
 **展開多層子目錄**  
 展開目前選取的項目下的所有層級。  
  
 **全部展開**  
 展開並顯示系統檢視中的所有層級。  
  
 **摺疊**  
 摺疊目前選取的項目下的所有層級。
