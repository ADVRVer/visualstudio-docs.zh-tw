# <a name="tree-menu-commands"></a>樹狀結構功能表命令
**樹狀**功能表提供命令來展開或摺疊檢視。  
  
 **展開一層**  
 會顯示下的一個層級擴充系統檢視中目前選取的項目。  
  
 **展開分支**  
 展開目前選取的項目下方的所有層級。  
  
 **全部展開**  
 會展開並顯示在系統檢視中的所有層級。  
  
 **摺疊**  
 摺疊目前選取的項目下方的所有層級。
