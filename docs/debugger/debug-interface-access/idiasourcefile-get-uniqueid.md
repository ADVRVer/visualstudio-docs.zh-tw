# <a name="idiasourcefilegetuniqueid"></a>IDiaSourceFile::get_uniqueId
擷取是唯一的這個映像的簡單的整數值。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_uniqueId (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回簡單的整數索引鍵的值是唯一的這個映像。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 比較索引鍵，而不是字串可加速行數處理。
