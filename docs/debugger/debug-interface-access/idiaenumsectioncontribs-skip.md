# <a name="idiaenumsectioncontribsskip"></a>IDiaEnumSectionContribs::Skip
略過指定的數目的列舉順序中的區段比重。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Skip(   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 `celt`  
 [in]略過將列舉序列中的區段比重數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`如果沒有更多的區段比重略過。
