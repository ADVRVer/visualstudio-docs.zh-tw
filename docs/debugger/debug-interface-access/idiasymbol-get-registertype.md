# <a name="idiasymbolgetregistertype"></a>IDiaSymbol::get_registerType
擷取註冊型別。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_registerType(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`保存暫存器類型。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
