# IDiaEnumInjectedSources::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

將列舉型別序列重設至開頭。  
  
## 語法  
  
```  
HRESULT Reset();  
```  
  
## 傳回值  
 傳回 `S_OK`。
