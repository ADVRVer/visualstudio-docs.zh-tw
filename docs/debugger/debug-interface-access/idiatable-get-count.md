# <a name="idiatablegetcount"></a>IDiaTable::get_Count
擷取資料表中的項目數目。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回資料表中的項目數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。
