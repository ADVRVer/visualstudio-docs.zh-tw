# <a name="idiaframedatagetvirtualaddress"></a>IDiaFrameData::get_virtualAddress
擷取框架的程式碼的虛擬位址 (VA)。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_virtualAddress (   
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回框架的程式碼的虛擬位址。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
