# <a name="idiainjectedsourcegetfilename"></a>IDiaInjectedSource::get_filename
擷取來源的檔案名稱。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_filename (   
   BSTR* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 pRetVal  
 [out]傳回來源的檔案名稱。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
