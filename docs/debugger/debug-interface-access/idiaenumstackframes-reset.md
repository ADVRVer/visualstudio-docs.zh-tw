# IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

將列舉型別 \(Enumeration\) 序列重設到開頭。  
  
## 語法  
  
```cpp#  
HRESULT Reset();  
```  
  
## 傳回值  
 傳回 `S_OK`。
