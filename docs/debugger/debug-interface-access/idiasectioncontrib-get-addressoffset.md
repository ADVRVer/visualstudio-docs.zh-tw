# <a name="idiasectioncontribgetaddressoffset"></a>IDiaSectionContrib::get_addressOffset
擷取比重的位址位移的一部分。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_addressOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回的比重位址的位移的部分。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
