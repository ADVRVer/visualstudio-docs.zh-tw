# 說明功能表命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[**說明**\] 功能表提供開啟說明檔的命令。  
  
 **Index**  
 開啟 Spy\+\+ 說明的內容畫面。  
  
 **使用說明**  
 開啟說明檔，其中描述如何使用說明。  
  
 **關於 Spy\+\+**  
 顯示 Spy\+\+ 的軟體版本和著作權資訊。
