# <a name="run-uwp-apps-from-visual-studio"></a>從 Visual Studio 執行 UWP 應用程式
UWP 應用程式上執行廣泛的裝置，以及 Visual Studio 提供您數種方式執行並與這些應用程式互動。 "F5" 體驗已獲增強，可在開發期間輕鬆地偵錯、分析和測試應用程式。
