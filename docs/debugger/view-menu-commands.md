# 檢視功能表命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[**檢視**\] 功能表提供控制 Spy\+\+ 使用者介面的命令。  
  
 **Font**  
 開啟 \[**字型**\] 對話方塊，您可以在此選取要用於目前 Spy\+\+ 檢視的字型。  
  
 **Toolbar**  
 隱藏或顯示工具列。  
  
 **Status Bar**  
 隱藏或顯示狀態列。  
  
 **屬性**  
 開啟作用中檢視的 \[**屬性**\] 對話方塊。
