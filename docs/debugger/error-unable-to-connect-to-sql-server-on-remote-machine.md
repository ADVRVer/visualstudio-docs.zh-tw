# 錯誤：無法連接至遠端電腦的 SQL Server
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

無法連接至遠端電腦 *name* 上的 SQL Server。  存取遭拒。  請確認您已在遠端電腦中安裝遠端偵錯工具。  如果遠端電腦不在網域中，或者 Visual Studio 是以本機帳戶執行，則遠端電腦必須具有一個與本機帳戶相同使用者名稱的帳戶和密碼。  
  
### 若要修正這個錯誤  
  
-   請參閱[遠端偵錯](../debugger/remote-debugging.md)。
