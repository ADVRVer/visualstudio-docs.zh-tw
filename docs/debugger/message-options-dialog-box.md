# 訊息選項對話方塊
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

使用這個對話方塊可以選取哪些訊息列在[訊息檢視](../debugger/messages-view.md)中。  若要顯示這個對話方塊，請從 \[**Spy**\] 功能表中選擇 \[**記錄檔訊息**\]。  
  
## 在本節中  
 [訊息選項對話方塊、視窗索引標籤](../debugger/windows-tab-message-options-dialog-box.md)  
 用來選取要列出之訊息的類型。  包括 \[搜尋工具\]。  
  
 [訊息選項對話方塊、訊息索引標籤](../debugger/messages-tab-message-options-dialog-box.md)  
 用來選取要檢視的訊息類型。  
  
 [訊息選項對話方塊、輸出索引標籤](../debugger/output-tab-message-options-dialog-box.md)  
 用來指定要顯示的訊息資料。
