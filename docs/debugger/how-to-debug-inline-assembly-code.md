# 如何：偵錯內嵌組譯程式碼
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

> [!NOTE]
>  根據目前使用的設定與版本，您所看到的對話方塊與功能表命令可能會與 \[說明\] 中所描述的不同。  若要變更設定，請從 \[工具\] 功能表中選取 \[匯入和匯出設定\]。  如需詳細資訊，請參閱 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-tw/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
 偵錯工具提供兩種視窗來偵錯內嵌組譯程式碼 \(Inline Assembly Code\)：\[**反組譯碼**\] 視窗和 \[**暫存器**\] 視窗。  
  
## 程序  
  
#### 若要偵錯內嵌組譯程式碼  
  
1.  使用 \[**反組譯碼**\] 視窗來檢視組譯碼指令。  
  
2.  使用 \[**暫存器**\] 視窗來檢視暫存器內容。
