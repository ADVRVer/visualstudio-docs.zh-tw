# How to: Change Fonts
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以變更 Spy\+\+ 視窗的字型、字型樣式和字型大小。  
  
### 若要變更字型選項  
  
1.  選擇 \[**檢視**\] 功能表中的 \[**字型**\]。  
  
2.  選擇字型、字型樣式和字型大小。  
  
3.  按一下 \[**確定**\]。  
  
 選取 \[**將字型存成預設值**\]，使未來所有的 Spy\+\+ 視窗都使用這個字型。
