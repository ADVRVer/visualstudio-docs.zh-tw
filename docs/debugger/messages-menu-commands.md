# 訊息功能表命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

\[**訊息**\] 功能表提供控制訊息檢視的命令。  
  
 **開始\/停止記錄**  
 開始或停止訊息檢視。  
  
 **記錄選項**  
 開啟 \[**訊息選項**\] 對話方塊，您可以在此指定訊息搜尋準則。  
  
 **清除記錄檔**  
 清除作用中 \[訊息\] 檢視的內容。
