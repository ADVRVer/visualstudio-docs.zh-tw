# MSBuild 錯誤 MSB1039
**MSB1039: 指定工具組的版本。**  
  
 使用 **\/ToolsVersion** 切換時，未指定值。  
  
### 若要更正這個錯誤  
  
-   當您使用 **\/ToolsVersion** 切換時，請確定指定了有效、非空值的 `ToolsVersion`。
