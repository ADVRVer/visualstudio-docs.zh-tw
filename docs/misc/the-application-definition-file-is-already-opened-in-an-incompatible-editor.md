# 應用程式定義檔已在不相容的編輯器中開啟
**此應用程式定義檔案已經在不相容的編輯器中開啟。  請先關閉其他編輯器，然後再重新載入此專案屬性頁。**  
  
 當您嘗試開啟已在另一個編輯器中開啟的檔案時，就會發生這個錯誤。  
  
### 若要更正這個錯誤  
  
1.  關閉另一個編輯器。  
  
2.  關閉專案設計工具。  
  
3.  按一下 \[**專案**\] 功能表上的 \[**屬性**\]，重新開啟專案設計工具。
