# <a name="servers-visual-studio-sdk"></a>伺服器 (Visual Studio SDK)
偵錯工具就架構而言，**伺服器**:  
  
-   為連接埠和通訊埠供應商的容器，用來與工作階段偵錯管理員 (SDM) 通訊連接埠和通訊埠供應商和偵錯引擎。  
  
-   可以識別本身的名稱，並列舉其連接埠和通訊埠供應商。  
  
-   由[IDebugCoreServer2](../../extensibility/debugger/reference/idebugcoreserver2.md)只 Visual Studio （Visual Studio 執行的每個執行個體的伺服器的一個執行個體） 所實作的介面。
