# <a name="process-debug-manager"></a>處理序偵錯管理員
處理序偵錯管理員 (PDM) 是管理程式的 Visual Studio 的元件和處理程序，讓它們可以在工作階段偵錯管理員和偵錯引擎。  
  
 PDM 管理可偵錯的所有處理序。 要進行偵錯程式必須向 PDM。 這是在啟動程式時，透過連接埠或偵錯引擎。
