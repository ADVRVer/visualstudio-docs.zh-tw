# <a name="idebugportpickersetsite"></a>IDebugPortPicker::SetSite
設定服務提供者。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetSite(  
   IServiceProvider * pSP  
);  
```  
  
```csharp  
public int SetSite(  
   IServiceProvider pSP  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pSP`  
 [in]服務提供者的介面參考。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 在呼叫其他任何方法之前，會呼叫這個方法。
