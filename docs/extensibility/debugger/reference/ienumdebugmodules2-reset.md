# <a name="ienumdebugmodules2reset"></a>IEnumDebugModules2::Reset
將列舉型別重設為第一個項目。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Reset(  
   void  
);  
```  
  
```c#  
int Reset();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 呼叫這個方法是，在下次呼叫之後[下一步](../../../extensibility/debugger/reference/ienumdebugmodules2-next.md)方法會傳回第一個項目，列舉型別。
