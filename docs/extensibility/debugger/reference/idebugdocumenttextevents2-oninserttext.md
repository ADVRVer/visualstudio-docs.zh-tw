# IDebugDocumentTextEvents2::onInsertText
[!INCLUDE[vs2017banner](../../../code-quality/includes/vs2017banner.md)]

告知偵錯封裝文字已插入到文件。  
  
## 語法  
  
```cpp#  
HRESULT onInsert(   
   TEXT_POSITION pos,  
   DWORD         dwNumToInsert  
);  
```  
  
```c#  
int onInsert(   
   enum_TEXT_POSITION pos,  
   uint               dwNumToInsert  
);  
```  
  
#### 參數  
 `pos`  
 \[in\]A [TEXT\_POSITION](../../../extensibility/debugger/reference/text-position.md)結構，表示文字插入的位置。  
  
 `dwNumToInsert`  
 \[in\]指定的文字插入的字元數目。  
  
## 傳回值  
 如果成功的話，會傳回`S_OK`。 否則，會傳回錯誤碼。
