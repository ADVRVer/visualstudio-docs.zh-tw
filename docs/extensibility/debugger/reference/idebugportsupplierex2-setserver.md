# <a name="idebugportsupplierex2setserver"></a>IDebugPortSupplierEx2::SetServer
設定連接埠供應商的核心伺服器。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetServer(  
   IDebugCoreServer2* pServer  
);  
```  
  
```csharp  
int SetServer(  
   IDebugCoreServer2 pServer  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pServer`  
 若要設定連接埠供應商的核心伺服器。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回錯誤碼。
