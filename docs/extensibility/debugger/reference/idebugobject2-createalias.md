# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
建立的唯一識別碼或此物件的別名，或傳回現有的別名。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT CreateAlias(  
   IDebugAlias** ppAlias  
);  
```  
  
```csharp  
int CreateAlias(  
   out IDebugAlias ppAlias  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppAlias`  
 [out]新的 （或現有的） 別名。  
  
## <a name="return-value"></a>傳回值  
 如果成功，會傳回 S_OK;反之則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 別名是代表特定的物件，而物件是在記憶體中的標籤。
