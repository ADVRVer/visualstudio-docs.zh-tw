# <a name="constructorenum"></a>CONSTRUCTOR_ENUM
選取不同類型的建構函式。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum ConstructorMatchOptions {   
   crAll       = 0,  
   crNonStatic = 1,  
   crStatic    = 2  
} CONSTRUCTOR_ENUM;  
```  
  
```csharp  
public enum ConstructorMatchOptions {   
   crAll       = 0,  
   crNonStatic = 1,  
   crStatic    = 2  
};  
```  
  
## <a name="members"></a>成員  
 crAll  
 選取所有建構函式。  
  
 crNonStatic  
 選取非靜態建構函式。  
  
 crStatic  
 選取 靜態建構函式。
