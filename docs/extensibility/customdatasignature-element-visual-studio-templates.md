# CustomDataSignature 項目 (Visual Studio 範本)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

指定要找到自訂資料的文字簽章。  
  
## 語法  
  
```  
<CustomDataSignature>"string"</CustomDataSignature>  
```  
  
## 屬性和項目  
 下列章節會說明屬性、子項目和父項目。  
  
### 屬性  
 無。  
  
### 子項目  
 無。  
  
### 父項目  
  
|元素|描述|  
|--------|--------|  
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|必要項目。<br /><br /> 為範本分類，並定義在 \[**新增專案**\] 或 \[**加入新項目**\] 對話方塊中範本的顯示方式。|  
  
## 文字值  
 需要文字值。  
  
 此文字是具有尋找自訂資料所需之文字簽章的字串。  
  
## 備註  
 `CustomDataSignature` 是選擇性項目。
