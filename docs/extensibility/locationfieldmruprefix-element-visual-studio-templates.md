# LocationFieldMRUPrefix 項目 (Visual Studio 範本)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

在 \[**新增專案**\] 和 \[**加入新項目**\] 對話方塊中指定最近使用的 \(MRU\) 路徑。  
  
## 語法  
  
```  
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>  
```  
  
## 屬性和項目  
 下列章節會說明屬性、子項目和父項目。  
  
### 屬性  
 無。  
  
### 子項目  
 無。  
  
### 父項目  
  
|元素|描述|  
|--------|--------|  
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|為範本分類，並定義在 \[**新增專案**\] 或 \[**加入新項目**\] 對話方塊中範本的顯示方式。|  
  
## 備註  
 這個項目只能用於透過 [!INCLUDE[vsipprvsip](../extensibility/includes/vsipprvsip_md.md)] 產生的範本。
