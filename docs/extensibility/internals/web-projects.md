# <a name="web-projects"></a>Web 專案
Web 專案系統會建立 Web 專案的專案系統。 Web 專案，依序建立 Web 應用程式。 您可以使用傳統的程式設計語言，例如網頁的程式設計[!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)]或[!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]，以收集和處理來自使用者的資訊、 將它儲存在資料庫中，等等。 使用相關聯的程式碼的網頁會呼叫智慧的網頁。
