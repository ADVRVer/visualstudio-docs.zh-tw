# 應用程式設定檔案中的連接屬性遺漏或不正確
應用程式設定檔案中的連接屬性遺漏或不正確。改用來自 .dbml 檔案的連接字串代替。  
  
 .dbml 檔案參考了應用程式設定檔案中所沒有的連接字串。這是告知性訊息，當您按一下 \[**確定**\] 時就會自動建立連接字串設定。  
  
### 若要回應這個訊息  
  
-   按一下 \[**確定**\]。.dbml 檔案中所含的連接資訊會加入至應用程式設定。
