# 無法建立關聯 &lt;關聯名稱&gt; - 屬性類型不符
無法建立關聯 \< 關聯名稱> -屬性類型不相符。 屬性沒有相符的型別︰ \< 屬性名稱>。  
  
 關聯是由所選 **關聯屬性** 中 **關聯編輯器** 對話方塊。 關聯兩端屬性的資料型別必須相同。  
  
 訊息中所列屬性的資料型別不同。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  檢查訊息，並記下訊息中指定的屬性。  
  
2.  按一下 [ **確定** 以關閉對話方塊。  
  
3.  檢查 **關聯屬性** 並選取相同的資料類型的屬性。  
  
4.  按一下 [確定]。
