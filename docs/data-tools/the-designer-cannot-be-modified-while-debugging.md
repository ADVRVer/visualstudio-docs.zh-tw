# 設計工具在偵錯時無法修改
如果應用程式是以偵錯模式執行，則嘗試修改 O\/R Designer 上的項目時，就會出現這則訊息。如果應用程式是以偵錯模式執行，則 O\/R Designer 會是唯讀的。  
  
### 若要更正這個錯誤  
  
-   按一下 \[**偵錯**\] 功能表上的 \[**停止偵錯**\]。  
  
     應用程式會停止偵錯，而 O\/R Designer 中的項目會成為可修改的狀態。
