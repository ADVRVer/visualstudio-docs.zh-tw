# 透過概念模型查詢資料
當您使用概念模型來描述資料庫中的資料時，可以透過物件查詢資料，而不需要在資料庫結構描述和物件模型之間來回轉譯。  
  
 您可以在 WCF 資料服務應用程式中使用概念模型。  下列主題說明如何透過概念模型查詢資料。
