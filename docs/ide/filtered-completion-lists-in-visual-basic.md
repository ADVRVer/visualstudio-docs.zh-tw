# Visual Basic 中已篩選的完成清單
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

在 Visual Basic 中，IntelliSense 完成清單在靠近清單底部的地方有兩個索引標籤控制項 \(Tab Control\)。  預設選取的 \[**通用**\] 索引標籤會顯示最常用來完成撰寫中之陳述式的項目。  \[**全部**\] 索引標籤則會顯示所有可供自動完成使用的項目，也包括 \[**通用**\] 索引標籤上的項目。
