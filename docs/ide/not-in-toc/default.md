# F1 說明

您似乎錯誤地來到此頁面。 如果您在尋找鍵盤或 API 名稱的 F1 說明，按下 **F1**之前，請務必確認游標位在文字內或在文字的開頭。 如果還是無法運作，請嘗試在 https://docs.microsoft.com 上搜尋。

如果您要搜尋文字編輯器本身的說明，請參閱[在編輯器中撰寫程式碼](../../ide/writing-code-in-the-code-and-text-editor.md)。
