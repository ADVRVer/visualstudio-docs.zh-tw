# <a name="locate-information"></a>尋找資訊
說明檢視器會提供幾種不同的方法，可用來找出 [說明] 中的資訊。 根據您所尋找的資訊類型，某些策略可能會比其他策略更有效率。  
  
|如需以下相關資訊|請嘗試|  
|----------------------------|---------|  
|說明檢視器使用者介面的元素|F1 鍵|  
|與目前的主題位於相同目錄區域中的其他主題|工具列上的 [顯示內容中的主題] 按鈕|  
|功能或技術|索引|  
|語言元素，例如方法和事件|搜尋|
