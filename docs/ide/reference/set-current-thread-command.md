# 設定目前執行緒命令
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

將指定的執行緒設定為目前的執行緒。  
  
## 語法  
  
```  
Debug.SetCurrentThread index  
```  
  
## 引數  
 `index`  
 必要項。  依其索引選取執行緒。  
  
## 範例  
  
```  
>Debug.SetCurrentThread 1  
```
