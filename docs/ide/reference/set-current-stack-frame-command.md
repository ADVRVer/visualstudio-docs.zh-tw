# 設定目前堆疊框架命令
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

允許您設定特定的堆疊框架。  
  
## 語法  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## 引數  
 `index`  
 必要項。  依照其索引選取堆疊框架。  
  
## 範例  
  
```  
>Debug.SetCurrentStackFrame 1  
```
