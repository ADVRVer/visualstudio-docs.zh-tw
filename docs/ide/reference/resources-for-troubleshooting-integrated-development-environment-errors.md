# Resources for Troubleshooting Integrated Development Environment Errors
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

並非所有的錯誤訊息都有專屬的相關說明主題。  如果錯誤訊息中的資訊未幫助您解決問題，您可以查詢其他資源，例如知識庫 \(Knowledge Base\) 文件、開發人員社群或產品支援。  
  
## 知識庫文件  
 您可以在線上知識庫 \(KB\) 搜尋有關產品問題的文章。  並非所有的問題都有對應的知識庫文章，不過有許多客戶都遇到過的錯誤通常會有文件記錄。  您可以存取上的知識庫文章[Microsoft 支援](http://go.microsoft.com/fwlink/?linkid=1782)的 Web 站台。  
  
## 開發人員社群  
 在論壇中您可以和其他開發人員以及 Microsoft 員工互動。  如果您所遇到的錯誤找不到解決方式，您可以將該錯誤的問題張貼到論壇上。  您也可以搜尋新聞群組，查看是否有其他人已張貼相同的問題。  
  
 您可以存取論壇、 網誌、 聊天室、 及其他資源上[Microsoft 技術社群](完整.嗎？LinkID%20=%20144922)的 Web 站台。  
  
## 產品支援  
 如果您嘗試其他的資源之後，仍然有疑問，您可以連絡 Microsoft 支援服務的 ¡ a 請造訪[Microsoft 支援](http://go.microsoft.com/fwlink/?linkid=1782)的 Web 站台。  如需您所在地的產品支援資訊，請參閱[告訴我們](../../ide/talk-to-us.md)。
