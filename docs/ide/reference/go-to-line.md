# 移至行
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

此對話方塊允許您移至使用中文件裡的特定行。  若要存取此對話方塊，請開啟要編輯的文件，然後選取 \[**編輯**\] 功能表上的 \[**移至**\]。  
  
 行號 \(1 \- `n`\)  
 允許您在要移動的使用中文件裡輸入行號。  輸入的號碼必須介於 1 和 `n` \(即目前文件中的行數\) 之間。
