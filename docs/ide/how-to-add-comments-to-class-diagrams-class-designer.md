# How to: Add Comments to Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以使用註解圖案為類別圖表加註。  註解圖案具備可以輸入文字的 \[**Text**\] 屬性。  註解圖案只會存在圖表介面上，而不會出現在程式碼中。  
  
 註解位於 \[**類別設計工具**\] 的 \[**類別圖表**\] 檢視上；如果您在同一個專案中開啟第二個類別圖表，就不會看到在第一個檢視中建立的註解。  如果刪除圖表，也會一併刪除內含的所有註解。  
  
 您可以調整註解圖案的大小，但是不能變更背景顏色、字型或字型大小等其他外觀。  
  
### 若要加入註解  
  
1.  從 \[**類別設計工具工具箱**\] 將 \[Comment\] 拖曳至類別圖表。  
  
2.  在圖表上按一下新的註解圖案，然後輸入您想要的文字。
