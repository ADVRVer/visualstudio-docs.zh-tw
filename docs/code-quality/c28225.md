# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28225: 沒有看到在註釋裡所需正確數目的參數  
  
 這則警告表示命名的註釋與不正確數目的參數一起使用。  標準巨集被作為註釋使用的情況是不應該發生的，這項警告在一般用法中顯示是不被預期的。
