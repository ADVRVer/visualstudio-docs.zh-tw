# C6296
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6296: 定義錯誤的 for 迴圈: 僅執行迴圈主體一次  
  
 這項警告指出 for 迴圈可能不會如預期地運作。  當索引未帶正負號且迴圈是從零向下計數時，則其主體僅會執行一次。  
  
## 範例  
 下列程式碼將產生出這個警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正這則警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
