# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28286: 函式結尾附近發生語法錯誤  
  
 程式碼分析工具會在可能錯誤的模型檔案時報告這個警告。  某些原始程式檔錯誤也會發生這種錯誤。
