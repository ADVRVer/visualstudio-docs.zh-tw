# C28196
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28196: 未滿足需求。\(運算式未評估為 true\)。  
  
 這則警告表示已分析之函式的函式原型有 `__notnull`、 `__null` 或 `__drv_valueIs` 在 `_Out_` 參數或傳回值，不過，傳回的值與的註釋不一致。
