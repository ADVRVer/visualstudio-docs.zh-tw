# C28210
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 28210: `_On_failure_` 內容的註釋不能在明確 pre context 中  
  
 附註 `_On_failure_` 必須明確或隱含地在 `__post` 內容表示，也就是說要在函式傳回後被套用。使用 `_drv_out` 確保這一點。
