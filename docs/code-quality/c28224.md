# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28224: 附註需要參數。  
  
 這則警告表示名為附註使用，而不使用參數，而且必須至少有一個參數。  如果標準巨集被做為附註是不可能的，這項警告不被預期在正常使用中顯示。
