# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28272: 函式與參數在檢查時的註釋與函式宣告不一致。  
  
 這則警告指出錯誤標記法，而不是在程式碼分析。   出現在函式定義的附註與出現在宣告的型別不一致。  應該解決兩個附註比對。
