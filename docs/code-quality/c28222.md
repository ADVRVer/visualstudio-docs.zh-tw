# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 28222:\_Yes\_、為附註預期的 \_No\_ 或 \_Maybe\_  
  
 這則警告表示附註中的參數必須是識別項`_Yes_``_No_`, or `_Maybe_`，和其他符號所遇到的。  這通常是指出一個不正確編碼的附註巨集。
