# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28232: pre、post 或 deref 未套用至任何附註。  
  
 這則警告表示 `_Pre_`、 `_Post_`或 `_Deref_` 運算子出現在附註運算式，但是沒有後續函式附註;修飾詞會被忽略，但是，這表示不正確編碼地的附註。
