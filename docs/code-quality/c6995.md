# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告C6995: 無法儲存 XML 記錄檔。  
  
 這則警告指出程式碼分析工具無法建立 Defect 記錄，是程式碼分析的輸出。  
  
 這個錯誤可能表示磁碟錯誤或表示您沒有權限可以在指定的目錄中的檔案。
