# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28160:錯誤附註  
  
 當遇到 `__drv_error` 附註，報告這個警告。  這個附註用來旗標應該修正的程式設計作法，且可與 `__drv_when` 附註表示參數的特定組合。
