# C28287
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28287: 函式的 \_At\_\(\) 附註中有語法錯誤 \(無效的參數名稱\)。  
  
 當使用 `SAL_at``__drv_at`標記法而且參數運算式不能在目前內容中解譯，程式碼剖析工具會報告這個警告。  使用了不正確的參數或成員名稱，這可能包括拼錯「Return」或「this」關鍵字。
