# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28223: 未找到預期附註、參數的語彙基元\/識別項。  
  
 這則警告表示附註中的參數必須是識別項，和其他符號所遇到的。  這通常是指出一個不正確編碼的附註巨集。  使用在這個位置的 C 或 C\+\+ 關鍵字會產生這個錯誤。
