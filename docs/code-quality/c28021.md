# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28021: 以\<anno\>標記附註的參數\<param\>，必須是指標。  
  
 當標註物件不是指標型別時，會回報警告。  這個附註無法使用 `void` 或整數類資料型別。
