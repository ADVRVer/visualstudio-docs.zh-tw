# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1252: 在外掛程式之間循環或遺失的相依性: 需要 GUID  
  
 若有內部錯誤和外掛程式相依性時，程式碼分析工具會報告這個警告，而非在進行分析程式碼時報告。
