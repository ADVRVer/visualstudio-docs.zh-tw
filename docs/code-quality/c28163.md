# C28163
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28163: 函式絕不能從 try\/except 區塊中呼叫:  
  
 這個警告報告，當函式在 `try/except` 區塊在 `try/except` 區塊不應該將找到的是型別時。  程式碼分析工具找到呼叫的函式會在 `try/except` 區塊中至少有一個路徑。
