# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28236: 註釋不能用在運算式中。  
  
 這則警告表示有嘗試在運算式內容宣告函式作為一種標註釋。  標準巨集被作為註釋使用的情況是不應該發生的，這項警告在一般用法中顯示是不被預期的。
