# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28020: 這個呼叫的運算式不是 true。  
  
 決定清單的 \_Satisfies\_ 運算式不為 true 時，會產生這個警告。  這通常表示無效的參數。  
  
 如果它在函式宣告中發生，則附註表示不在可能的情況。
