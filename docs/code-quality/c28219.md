# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28219: 註釋中標註的參數需要列舉的成員  
  
 為附註的參數必須是具名 `enum` 型別成員; 其他符號遇到，使用這個 `enum` 型別的成員。  這通常是指出一個不正確編碼的附註巨集。
