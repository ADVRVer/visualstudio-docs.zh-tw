# <a name="c28219"></a>C28219
警告 C28219： 註釋需要列舉的成員註解中的參數  
  
 註釋的參數必須是具名的成員`enum`遇到類型，以及某些其他符號，則使用該成員`enum`型別。 這通常表示編碼錯誤的註解巨集。
