# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28351: 註釋\<annotation\>描述條件中不可以使用動態值 \(變數\)。  
  
 通常這個警告會在附註在 C\/C\+\+ 型別被檢查被套用時被產生。  
  
 `_When_`中運算式的計算結果應為常數。  `_When_` 會被忽略。
