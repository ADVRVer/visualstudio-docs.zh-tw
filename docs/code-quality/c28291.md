# C28291
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28291: 對函式 \<function\> 參數 \<number\> 的取值層級 0 使用 post null\/notnull 沒有意義  
  
 當取值層級零參數的後置條件指定具有 null\/非 null 的屬性時，程式碼剖析工具報告這個警告。  因為在取值層級零的值無法變更，就會發生這個錯誤。
