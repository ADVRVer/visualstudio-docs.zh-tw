# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28301: \<function\> 的第一個宣告中沒有註釋。\<note1\> 請參閱 \<filename\>\(\<line\>\)。\<note2\>  
  
 在附註中找不到所指定函式的第一個宣告，會回報警告。
