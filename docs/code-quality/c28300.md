# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28300: \<parameter\_name\>: 不相容的型別運算元運算子的 \<operator\_name\>  
  
 當運算元型別參數與運算子相容時，會產生此警告。
