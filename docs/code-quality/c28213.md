# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28213: `_Use_decl_annotations_` 註釋必須不需修改就能用來參考預先宣告。\<why\>  
  
 當使用先前宣告的參考，但沒有使用必要的 `_Use_decl_annotations_` 附註時，此警告將會報告。
