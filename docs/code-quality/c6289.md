# <a name="c6289"></a>C6289
警告 C6289： 不正確的運算子： 互斥的情況 &#124; &#124;是永遠為非零的常數。 您想要使用 （& s) （& s) 而嗎？  
  
 這個警告表示會針對兩個不同的常數測試變數中的測試運算式，結果會取決於符合其中一個條件。 這一律會評估為 true。  
  
 此問題通常因使用`||`取代`&&`，但也可能會因使用`!=`其中`==`設計的目的。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int x)  
{  
  if ((x != 1) || (x != 3))  
  {  
    // code   
  }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f(int x)  
{  
  if ((x != 1) && (x != 3))  
  {  
    // code   
  }  
}  
  
/* or */  
void f(int x)  
{  
  if ((x == 1) || (x == 3))  
  {  
    // code   
  }  
}  
```
