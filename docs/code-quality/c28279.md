# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28279: 找到符號的 'begin'，但沒有相符的 'end'。  
  
 附註語言支援的開始和結束`{` 和`}`在 C\) 建構和配對取得不對稱。   如果使用巨集，這個條件可以避開的。
