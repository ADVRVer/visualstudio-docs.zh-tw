# C28238
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28238: 註釋的參數具有一個以上的值，stringValue 和 longValue。  請使用 paramn\=xxx。  
  
 一個內部錯誤在 PREfast 模型檔案內發生。  這項警告在一般用法應該不會發生。
