# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28229: 註釋不支援參數。  
  
 當附註宣告但沒有參數時，這則警告表示附註使用搭配參數。  標準巨集被作為註釋使用的情況是不應該發生的，這項警告在一般用法中顯示是不被預期的。
