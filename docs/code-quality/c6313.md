# <a name="c6313"></a>C6313
警告 C6313： 不正確的運算子： 不能使用位元來測試值為零的旗標-和。 使用相等測試檢查有值為零的旗標  
  
 這個警告表示做為引數的位元提供常值為 0-和 (`&`) 測試內容中的運算子。 所產生的運算式是常數，並評估為 false。結果會不同於原先想要的。  
  
 這通常因為使用位元集，以測試是否有值為零的旗標。 若要測試的值為零的旗標，等號比較測試必須，例如，使用執行`==`或`!=`。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#define FLAG 0  
  
void f(int Flags )  
{  
  if (Flags & FLAG)  
  {  
    // code  
  }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
#define FLAG 0  
  
void f(int Flags )  
{  
  if (Flags == FLAG)  
  {  
    // code  
  }  
}  
```
