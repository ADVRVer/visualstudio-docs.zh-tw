# <a name="c28211"></a>C28211
警告 C28211: sal_context 需要靜態內容名稱  
  
 這則警告表示所要的運算元`_Static_context_`註解必須是工具定義的內容的名稱。 在正常使用，應該永遠不會發生這個警告。
