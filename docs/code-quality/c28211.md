# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28211: SAL\_context 必須有靜態內容名稱。  
  
 這個警告表示在 `_Static_context_` 附註的運算元必須是一個工具定義之內容的名稱。  這項警告在正常使用應該不會發生。
