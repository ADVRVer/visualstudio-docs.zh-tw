# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28221: 用於附註的參數預期的字串運算式。  
  
 這則警告表示附註中的參數必須是識別項，以及其它種所遇到的型別。  這在正常使用通常會是不正確地編碼的附註巨集和預期不會顯示。
