# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28212: 附註預期指標運算式  
  
 這個警告表示附註的參數數字 \(附註的不是函式\) 必須是指標或陣列型別，但卻遇到了其他的型別。  附註需要被更正。
