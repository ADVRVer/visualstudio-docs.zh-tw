# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28220: 整數運算式為附註預期標記法的參數  
  
 這則警告表示附註中的參數必須是整數，而運算式則是遇到其他的型別。  這通常是指出一個不正確編碼的附註巨集。
