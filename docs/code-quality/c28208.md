# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28208: 函式 \<function\> 為以 \<file\>\(\<line\>\) 的不同的參數清單先前定義的。  有些分析工具會造成不正確的結果。  
  
 當函式的定義不符合其他發生時，這個警告報告。
