# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28228: 附註參數: 無法使用型別附註  
  
 這則警告表示參數不支援的型別。  附註只使用有限的型別做為參數。  標準巨集被作為註釋使用的情況是不應該發生的，這項警告在一般用法中顯示是不被預期的。
