# C28226
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28226: 附註不可同時是 PrimOp \(在目前宣告中\)。  
  
 這則警告先前宣告為 PrimOp，表示名為附註宣告為泛型的附註。  如果標準巨集被做為附註是不可能的，這項警告不被預期在正常使用中顯示。
