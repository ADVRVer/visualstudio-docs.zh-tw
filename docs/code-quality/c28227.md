# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28227: 註釋不能也是 PrimOp \(參閱之前的宣告\)  
  
 這則警告表示名為附註宣告為泛型的附註，以及先前宣告為 PrimOp。  標準巨集被作為註釋使用的情況是不應該發生的，這項警告在一般用法中顯示是不被預期的。
