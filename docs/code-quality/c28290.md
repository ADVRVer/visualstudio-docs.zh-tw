# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28290: 函式的註釋包含比實際參數數目還多的外部。  
  
 當函式之附註的參數時，的實際數目包含多個外部程式碼分析工具報告這個警告。
