# C28243
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28243: 函式 \(位於參數\) 的註釋需要執行的取值比實際註釋之類型允許的還多。  
  
 附註的 `__deref` 運算子數目大於參數型別定義的指標的層級數目。  藉由變更標記的數字或參考參數的指標標準更正這個警告。
