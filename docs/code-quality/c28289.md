# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

ReadableTo 或 WritableTo 沒有有限的規格做為參數。  
  
 當函式\/參數標記法 miscoded 如上所述時，程式碼分析工具報告這個警告。
