# <a name="c6515"></a>C6515
警告 C6515： 無效的註釋：\<名稱 > 屬性只能用於指標類型的值  
  
 這個警告表示用於指標的屬性已套用至非指標類型。 如需註解屬性的清單，請參閱[附註屬性](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
