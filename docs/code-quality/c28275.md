# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28275: \_Macro\_value\_ 的參數為 null。  
  
 這個警告表示模型檔案的內部錯誤，而不是在進行分析的程式碼。  *macroValue* 函式呼叫，而不使用參數。
