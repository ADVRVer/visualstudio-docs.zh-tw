# C1255
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C1255警告:外掛程式的 PCH 資料長度不正確。  
  
 當工具中有內部錯誤而不是在分析程式碼時，程式碼分析工具會回報這個警告。
