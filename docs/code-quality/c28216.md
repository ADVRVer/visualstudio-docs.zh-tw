# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28216: \_Check\_return\_ 附註只適用於之後狀況的特定函式參數。  
  
 `_Check_return_` 附註已經被應用於另一個除了 `__post` 的內容。它可能需要 `__post` \(或 `__drv_out`\) 修飾詞， 或者它可能被放置錯誤。
