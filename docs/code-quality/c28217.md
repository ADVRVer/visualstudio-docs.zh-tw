# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告C28217:對於函式，附註的參數數目與找到的不符。  
  
 附註在目前行中以及在訊息的列中不一致。  如果標準巨集被做為附註是不可能的，這項警告不被預期在正常使用中顯示。
