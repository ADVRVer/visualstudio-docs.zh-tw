# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告C28239: 參數的註釋具有兩個值，stringValue 或 longValue 以及 paramn\=xxx。  請只使用 paramn\=xxx。  
  
 一個內部錯誤在 PREfast 模型檔案內發生。  這項警告在一般用法應該不會發生。
