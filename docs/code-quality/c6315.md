# C6315
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6315: 不正確的作業順序: 位元 AND 的優先順序高於位元 OR。  加入括號來釐清目的。  
  
 這則警告表示測試內容中的運算式同時包含位元 AND \(`&`\) 和位元 OR \(       `|` \) 運算，但會造成常數，因為位元 OR 運算是最後進行的。  應該加入括號以釐清目的。  
  
## 範例  
 下列程式碼將產生出這個警告：  
  
```  
void f( int i )  
{  
  if ( i & 2 | 4 ) // warning  
  {  
    // code  
  }  
}  
```  
  
 若要更正這則警告，請加入括號，如下列程式碼所示：  
  
```  
void f( int i )  
{  
  if ( i & ( 2 | 4 ) )  
  {  
    // code  
  }  
}  
```
