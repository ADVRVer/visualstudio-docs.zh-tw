# C6510
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6510:無效的附註:「NullTerminated」屬性在項目是整數或指標型別的緩衝區只能使用: '\<函式\>' \<參數\>。  
  
 這個警告表示不正確地使用 **NullTerminated** 屬性\(結束於 `_z`\)。  只可在指標或陣列型別上使用這個屬性。  
  
## 範例  
 下列程式碼將產生出這個警告：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char x)  
{  
    // code ...  
}  
```  
  
 若要更正這則警告，請使用下列程式碼：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char * x)  
{  
    // code ...  
}  
```
