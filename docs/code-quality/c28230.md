# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28230: 參數的型別沒有成員。  
  
 這則警告表示附註的引數嘗試存取 `struct`， `class`，或 `union` 和具名的成員不存在。
