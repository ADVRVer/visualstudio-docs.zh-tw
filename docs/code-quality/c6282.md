# C6282
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6282: 不正確的運算子: 在布林內容中指派常數。  請考慮改用 '\=\='  
  
 這則警告表示在測試內容中偵測到將常數指派給變數。  在測試內容中將常數指派給變數幾乎都不正確。  將 `=` 替換為 `==`，或從測試內容中移除指派，以解除這則警告。  
  
## 範例  
 下列程式碼將產生出這個警告：  
  
```  
void f( int i )  
{  
   while (i = 5)  
   {  
   // code    
   }  
}  
```  
  
 若要更正這則警告，請使用下列程式碼：  
  
```  
void f( int i )  
{  
   while (i == 5)  
   {  
   // code    
   }  
}  
```
