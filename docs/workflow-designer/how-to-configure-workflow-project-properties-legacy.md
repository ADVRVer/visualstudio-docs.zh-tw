# HOW TO：設定工作流程專案屬性 (舊版)
依照下列步驟來檢視或修改與工作流程專案相關聯的屬性。  
  
### 若要檢視或修改工作流程專案屬性  
  
1.  如果看不到 \[**方案總管**\] 視窗，請選取 \[**檢視**\] 功能表上的 \[**方案總管**\]。  
  
2.  在 \[**方案總管**\] 中選取專案。  
  
3.  如果看不到 \[**屬性**\] 視窗，請選取 \[**檢視**\] 功能表上的 \[**屬性視窗**\]。  
  
4.  在專案 \[**屬性**\] 視窗中，檢視並修改您要變更的屬性。
