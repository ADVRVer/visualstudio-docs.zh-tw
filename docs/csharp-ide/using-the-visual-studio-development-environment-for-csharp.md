# <a name="using-the-visual-studio-development-environment-for-c"></a>若為 C# 中使用 Visual Studio 開發環境 #
Visual Studio 整合式的開發環境 (IDE) 是透過一般的使用者介面公開的開發工具的集合。 某些工具會與其他的 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 語言共用，而某些工具 (例如 C# 編譯器) 則是 Visual C# 特有的。 本主題提供最重要 Visual C# 工具的連結。
