# 如何：將功能新增至 SharePoint 方案
  當您建立 SharePoint 方案時，系統會將一個功能和一個套件加入至方案。  您可以加入更多功能項目，以組織不同類型的 SharePoint 檔案。  
  
### 若要透過方案總管加入功能  
  
1.  在 \[**方案總管**\] 中，開啟 \[**功能**\] 的捷徑功能表，然後選擇 \[**加入功能**\]。
