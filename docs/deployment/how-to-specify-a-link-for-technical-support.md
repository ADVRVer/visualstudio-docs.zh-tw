# 如何：指定技術支援的連結
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

發行 [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] 應用程式時，\[**支援 URL**\] 屬性會識別 Web 網頁或檔案共用，讓使用者前往這個位置，取得應用程式的詳細資訊。  這個屬性是選擇性；如果提供這個屬性，在應用程式項目 \[**新增或移除程式**\] 對話方塊中，就會顯示此 URL。  
  
 在 \[**專案設計工具**\] 的 \[**發行**\] 頁上，可以設定 \[**支援 URL**\] 屬性。  
  
### 若要指定支援 URL  
  
1.  在 \[**方案總管**\] 中選取專案之後，請在 \[**專案**\] 功能表上，按一下 \[**屬性**\]。  
  
2.  按一下 \[**發行**\] 索引標籤。  
  
3.  按一下 \[**選項**\] 按鈕，開啟 \[**發行選項**\] 對話方塊。  
  
4.  按一下 \[**描述**\]。  
  
5.  在 \[**支援 URL**\] 欄位中，輸入網站、Web 網頁或 UNC 共用的完整路徑。
