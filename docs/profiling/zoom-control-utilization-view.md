# 縮放控制 (使用率檢視)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

縮放控制項可用來放大和縮小 CPU 使用率圖表，讓您可以將焦點放在特別值得注意的區域。  這個控制項在檢視的中央放大。  因此放大前所要的區域應該置於中央。  
  
 您可以利用滑鼠指標對 CPU 使用率圖表或 GPU 活動圖表建立反白顯示的區域。  放開滑鼠按鈕時，這個檢視會放大選定的範圍。
