# 核心檢視邏輯核心
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

每個通道名稱代表受程式碼剖析系統上存在的其中一個邏輯核心。  命名方式會從 0 開始數到邏輯核心總數。  例如，四核心系統會從邏輯核心 0 數到邏輯核心 3。
