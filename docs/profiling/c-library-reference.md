> [!NOTE]
>  C\+\+ 語言標準針對程式庫等實作保留了以底線 \(`_`\) 字元為開頭之識別碼的用法。   請勿在您的程式碼裡使用以底線為開頭的名稱。  我們不保證名稱遵循這個慣例之程式碼項目的行為，而且未來的發行版本可能會變更。  因為這個原因，這類程式碼項目會從這個檔案忽略。
