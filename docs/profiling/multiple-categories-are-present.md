# 存在多個分類
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

已選取指定時間範圍內包含多種活動類型的區段。  請注意顯示兩個色彩，其中一個垂直堆疊於另一個之上。  這些色彩就是代表指定時間範圍內前兩個最頻繁的活動。  這兩個活動的頻率會彼此相對顯示為垂直列的三分之一或二分之一。  若要查看更多詳細資訊，請將執行緒檢視的這個區段放大。
