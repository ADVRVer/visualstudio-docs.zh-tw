# 執行緒就緒連接器
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

當您按一下封鎖區段來查看呼叫堆疊及其解決封鎖堆疊，執行緒就緒連接器也可能會出現。  如果解除封鎖事件發生在目前處理序的另一個執行緒上，執行緒就緒連接器會以視覺方式識別該執行緒以及讓封鎖的執行緒恢復執行的執行區段。
