# 可見時間表分析
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

「執行緒封鎖檢視」的作用中圖例可提供統計資訊和報告的連結。   當您放大、縮小、水平捲動、隱藏通道或顯示通道時，作用中圖例裡的數字會隨之變更，以反映檢視中的目前項目。  若要檢視圖例中某個項目的報告，請按一下該項目。
