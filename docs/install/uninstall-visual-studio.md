# <a name="uninstall-visual-studio"></a>解除安裝 Visual Studio
Visual Studio 是我們提供給開發人員的整合式生產力工具套件，本頁面將引導您完成 Visual Studio 的解除安裝作業。  

1.  在電腦上找到 Visual Studio 安裝程式。  

     例如，在執行 Windows 10 年度更新版的電腦上，按一下 [開始]，然後捲動至字母 **M**，您會在其中看到其列為 **Microsoft Visual Studio 安裝程式**。  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png)

2.  按一下以啟動安裝程式，按一下![詳細資料圖示](media/vs2017uninstall-UninstallIcon.png)圖示，然後按一下 [解除安裝]。  

     ![修改 Visual Studio 2017 RC：啟動或修改](media/vs2017uninstall-ChooseUninstallFromInstaller.png "修復或解除安裝 Visual Studio 2017 RC")  

3.  按一下 [確定] 確認您的選擇。

     ![修改 Visual Studio 2017 RC：啟動或修改](media/vs2017uninstall-UninstallConfirm.png "確認解除安裝 Visual Studio 2017 RC")  

如果您稍後改變心意並想重新安裝 Visual Studio 2017 RC，只要重新啟動 Visual Studio 安裝程式，然後從選擇畫面中按一下 [安裝] 即可。

> [!IMPORTANT]
> 雖然 Visual Studio 2017 RC 一般支援在生產環境中使用，但安裝 UI 中標示為「預覽」的這些工作負載和元件不支援在生產環境中使用。
