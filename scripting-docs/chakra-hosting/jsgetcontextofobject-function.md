# JsGetContextOfObject 函式
取得物件所屬的指令碼內容...  
  
## 語法  
  
```  
STDAPI_(JsErrorCode) JsGetContextOfObject(  
  _In_ JsValueRef object,  
  _Out_ JsContextRef *context  
);  
```  
  
#### 參數  
 `object`  
 要從中取得內容的物件。  
  
 `context`  
 物件所屬的內容。  
  
## 傳回值  
 如果作業成功，則為 `JsNoError` 碼，否則為失敗碼。  
  
## 備註  
 需要使用中指令碼內容。  
  
## 需求  
 **標頭：**jsrt.h
