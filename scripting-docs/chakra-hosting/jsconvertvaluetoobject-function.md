# JsConvertValueToObject 函式
使用標準 JavaScript 語意將值轉換為物件。  
  
## 語法  
  
```  
STDAPI_(JsErrorCode) JsConvertValueToObject(  
   _In_ JsValueRef value,  
   _Out_ JsValueRef *object  
);  
```  
  
#### 參數  
 `value`  
 要轉換的值。  
  
 `object`  
 轉換的值。  
  
## 傳回值  
 如果作業成功，則為 `JsNoError` 碼，否則為失敗碼。  
  
## 備註  
 需要使用中指令碼內容。  
  
## 需求  
 **標頭：**jsrt.h
