# JsIntToNumber 函式
透過 `int` 值來建立數字值。  
  
## 語法  
  
```  
STDAPI_(JsErrorCode) JsIntToNumber(  
   _In_ int intValue,  
   _Out_ JsValueRef *value  
);  
```  
  
#### 參數  
 `intValue`  
 要轉換成數字值的 `int`。  
  
 `value`  
 新的數字值。  
  
## 傳回值  
 如果作業成功，則為 `JsNoError` 碼，否則為失敗碼。  
  
## 備註  
 需要使用中指令碼內容。  
  
## 需求  
 **標頭：**jsrt.h
