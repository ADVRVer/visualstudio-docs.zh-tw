# JsValueRef Typedef
JavaScript 值的參考。  
  
## 語法  
  
```  
typedef JsRef JsValueRef;  
```  
  
## 備註  
 JavaScript 值是下列其中一種類型的值：未定義、Null、布林值、字串、數值或物件。
