# JsGetStringLength 函式
取得字串值的長度。  
  
## 語法  
  
```  
STDAPI_(JsErrorCode) JsGetStringLength(  
   _In_ JsValueRef stringValue,  
   _Out_ int *length  
);  
```  
  
#### 參數  
 `stringValue`  
 要取得其長度的字串值。  
  
 `length`  
 字串的長度。  
  
## 傳回值  
 如果作業成功，則為 `JsNoError` 碼，否則為失敗碼。  
  
## 備註  
 需要使用中指令碼內容。  
  
## 需求  
 **標頭：**jsrt.h
