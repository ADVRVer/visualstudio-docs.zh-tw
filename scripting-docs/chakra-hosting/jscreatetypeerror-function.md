# JsCreateTypeError 函式
建立新的 JavaScript TypeError Error 物件  
  
## 語法  
  
```  
STDAPI_(JsErrorCode) JsCreateTypeError(  
   _In_ JsValueRef message,  
   _Out_ JsValueRef *error  
);  
```  
  
#### 參數  
 `message`  
 Error 物件的訊息。  
  
 `error`  
 新的 Error 物件。  
  
## 傳回值  
 如果作業成功，則為 `JsNoError` 碼，否則為失敗碼。  
  
## 備註  
 需要使用中指令碼內容。  
  
## 需求  
 **標頭：**jsrt.h
