# 必須是 &#39;{&#39;
您並未包含標示函式主體開頭的左大括號。  組成函式主體的程式碼 \(即使只有一行\) 都必須包含在左大括號和右大括號之內。  請注意，在函式主體中使用大括號會比在迴圈中使用更為嚴格。  
  
### 若要更正這個錯誤  
  
-   在函式主體加入做為開頭標記的左大括號。
