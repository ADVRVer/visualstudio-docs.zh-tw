# 未定義的識別項
您嘗試使用 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 編譯器無法辨識的識別項。  每當您使用以下項目時，就會傳回未定義值：  
  
-   不存在的變數、  
  
-   已經宣告但是從未指派值的變數、  
  
-   不存在的物件屬性。  
  
### 若要更正這個錯誤  
  
-   以 **var** 陳述式 \(如 `var` x;\) 宣告變數。
