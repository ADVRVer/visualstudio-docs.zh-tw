# 未結束的字串常數 (JavaScript)
您的字串常數並未以結束的引號當做結尾。  字串常數必須置於一對引號內。  
  
> [!NOTE]
>  您可以使用成對的單引號或雙引號。  雙引號可以包含在由單引號包圍的字串內，而單引號可以包含在由雙引號包圍的字串內。  
  
### 若要更正這個錯誤  
  
-   在字串的結尾加入結束的引號。
