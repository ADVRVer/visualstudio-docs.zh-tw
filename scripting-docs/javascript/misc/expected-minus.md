# 必須是 &#39;/&#39;
您嘗試建立規則運算式常值，但並未包含其中一個斜線 \(\/\)。  就像是使用一對引號內的字元撰寫字串常值一樣，規則運算式常值會表示為一對斜線 \(\/\) 字元中的字元。  
  
### 若要更正這個錯誤  
  
-   插入結尾的正斜線，以標記規則運算式的結尾。
