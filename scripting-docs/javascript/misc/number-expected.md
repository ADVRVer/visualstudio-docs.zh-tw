# 必須是數字
您嘗試在 **Number** 型別以外的物件上叫用 **Number.prototype.toString** 或 **Number.prototype.valueOf** 方法。  這種叫用的物件必須是 **Number** 型別。  
  
### 若要更正這個錯誤  
  
-   只能在 **Number** 型別的物件上叫用 **Number.prototype.toString** 或 **Number.prototype.valueOf** 方法。
