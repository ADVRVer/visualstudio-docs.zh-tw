# 必須是常數
您嘗試在條件式編譯測試陳述式中使用 \(非條件式編譯\) 變數。  條件式編譯測試陳述式必須評估為常數。  
  
### 若要更正這個錯誤  
  
-   以常值取代變數。  
  
-   以條件式編譯變數取代變數。
