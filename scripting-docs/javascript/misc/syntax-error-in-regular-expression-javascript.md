# 規則運算式中的語法錯誤 (JavaScript)
您的搜尋字串結構違反一個或多個 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 規則運算式的文法規則。  
  
### 若要更正這個錯誤  
  
-   請確定規則運算式搜尋字串的結構有遵循 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 規則運算式語法。
