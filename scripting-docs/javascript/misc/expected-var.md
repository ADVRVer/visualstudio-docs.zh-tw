# 必須是 &#39;var&#39;
必須是 'var' \(ERRnoVar 1034\)  
  
 您在需要關鍵字 var 時未能加入此關鍵字。  
  
 如需 var 關鍵字的詳細資訊，請參閱 [var 陳述式](../../javascript/reference/var-statement-javascript.md)。
