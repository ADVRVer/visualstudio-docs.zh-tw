# 必須是 &#39;@end&#39;
您嘗試建立有條件編譯的程式碼區塊，但是結尾並未包含 **@end** 陳述式。  若要有條件地編譯 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 陳述式，可以將其放在 **@if\/@end** 區塊內。  
  
### 若要更正這個錯誤  
  
-   加入對應的 **@end** 陳述式。
