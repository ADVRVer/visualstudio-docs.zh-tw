# 必須是 &#39;]&#39;
您已參考某個陣列元素但是並未包含右方括號。  參考陣列元素的任何運算式都必須包含左右方括號。  
  
### 若要更正這個錯誤  
  
-   請將右方括號加入至參考陣列元素的運算式。
