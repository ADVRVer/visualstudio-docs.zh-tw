# 必須是 &#39;while&#39;
您並未在 `do...while` 迴圈中包含 `while` 條件。  **do** 陳述式在程式碼區塊結尾必須有對應的 `while` 測試。  
  
### 若要更正這個錯誤  
  
-   在右大括號後面包含 `while` 測試陳述式。
