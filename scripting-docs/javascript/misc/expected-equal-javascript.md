# 必須是 &#39;=&#39; (JavaScript)
您嘗試建立要搭配條件式編譯陳述式使用的變數，但是並未在變數與您想要指派給變數的值之間放置等號。  
  
### 若要更正這個錯誤  
  
-   加入等號。  例如：  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
