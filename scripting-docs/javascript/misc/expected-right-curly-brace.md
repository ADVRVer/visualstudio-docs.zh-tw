# 必須是 &#39;}&#39;
您未包含右邊大括號，此大括號會標示函式主體、迴圈、程式碼區塊或物件初始設定式的結尾。  只包含左大括號來標示迴圈主體的 **for** 迴圈為這個錯誤的一個例子。  
  
### 若要更正這個錯誤  
  
-   請加入右大括號來標示函式、迴圈、區塊或物件初始設定式的結尾。
