# 必須要有 JavaScript 物件
您嘗試將非 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 物件傳遞給必須是 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 物件的內建函式。  各種內建函式都需要 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 中定義的物件 \(主應用程式或類似控制項的外部元件所定義的物件則相反\)。  
  
### 若要更正這個錯誤  
  
-   請確定您以參數形式傳入的物件具有正確型別。
