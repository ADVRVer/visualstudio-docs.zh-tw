# 必須是字串
您嘗試針對型別不是 `String` 的物件叫用 **String.prototype.toString** 或 **String.prototype.valueOf** 方法。  這種叫用的目標物件必須是 `String` 型別。  
  
### 若要更正這個錯誤  
  
-   您只能針對型別是 `String` 的物件叫用 **String.prototype.toString** 或 **String.prototype.valueOf** 方法。
