# 必須是 &#39;;&#39;
您嘗試將多個陳述式放在同一行，但是並未以分號分隔這些陳述式。  分號是用來表示一段陳述式已結束。  雖然您可以將多個陳述式放在同一行，但是必須以分號分隔這些陳述式。  
  
### 若要更正這個錯誤  
  
-   在每個陳述式的結尾加上分號。  
  
-   確定您的函式呼叫都有適當地使用括弧。  
  
-   確定您並未忘記在 **for** 迴圈的標頭內加上分號。
