# 必須是 &#39;(&#39; (JavaScript)
您嘗試將運算式放在一組括號中，但是並未包含左括號。  某些運算式必須置於一組左括號和右括號中。  請注意下列範例中括弧的用法。  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### 若要更正這個錯誤  
  
-   請將左括號加入至評估運算式。
