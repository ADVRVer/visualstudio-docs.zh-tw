# 必須是識別項、字串或數字
您使用了不正確的常值語法宣告物件常值。  物件常值的屬性必須是識別項、字串或數字。  物件常值 \(也稱為「物件初始設定式」\) 是由逗號分隔的屬性:值配對清單所組成 \(全都在方括號內\)。  例如：  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### 若要更正這個錯誤  
  
-   請務必使用適當的常值語法。
