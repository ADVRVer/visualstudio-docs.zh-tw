# 必須是識別項 (JavaScript)
您在需要識別項的內容中使用了識別項以外的項目。  識別項可以是：  
  
-   變數、  
  
-   屬性、  
  
-   陣列  
  
-   或函式名稱。  
  
### 若要更正這個錯誤  
  
-   變更運算式，好讓識別項出現在等號的左邊。
