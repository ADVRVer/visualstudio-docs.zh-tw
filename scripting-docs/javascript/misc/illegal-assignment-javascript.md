# 無效的指派 (JavaScript)
您嘗試指派值給唯讀識別項。  您無法指派值給唯讀識別項。  例如，主應用程式定義的物件和外部 COM 物件。  
  
### 若要更正這個錯誤  
  
-   請勿指派值給唯讀識別項。
