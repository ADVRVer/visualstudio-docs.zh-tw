# 必須是 &#39;)&#39; (JavaScript)
您嘗試將運算式放在一組括弧中，但是並未包含右括弧。  某些運算式必須置於一組左括弧和右括弧中。  請注意下列範例中括弧的用法。  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### 若要更正這個錯誤  
  
-   請為評估運算式加上右括弧。
