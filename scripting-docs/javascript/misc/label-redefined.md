# 已重新定義標籤
您已經建立新的標籤，但是為此標籤提供了現有標籤的名稱。  標籤可用來標記程式碼區塊，不過標籤在指定的範圍中必須是唯一的。  
  
### 若要更正這個錯誤  
  
-   請確定您在程式中使用的所有標籤在其各自範圍內都是唯一的。
