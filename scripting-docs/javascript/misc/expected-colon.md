# 必須是 &#39;:&#39;
您嘗試使用三元的條件運算子建立運算式，但是並未在第二個和第三個運算元之間加入冒號。  三元 \(三個運算元\) 條件運算子需要在第二個 \(true\) 和第三個 \(false\) 運算元之間有冒號。  
  
### 若要更正這個錯誤  
  
-   請在第二個和第三個運算元之間插入冒號。
