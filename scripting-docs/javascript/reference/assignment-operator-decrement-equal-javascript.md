# 指派運算子 (=) (JavaScript)
指派值給變數。  
  
## 語法  
  
```  
  
result = expression  
```  
  
## 參數  
 `result`  
 任何變數。  
  
 `expression`  
 任何數值運算式。  
  
## 備註  
 \= 運算子的行為方式如同其他運算子，所以包含它的運算式具有值。  這表示您可以鏈結指派運算子，方法如下：`j = k = l = 0`。  在這個案例中，`j`、`k` 和 `l` 都等於零。
