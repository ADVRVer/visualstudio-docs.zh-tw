# size 屬性 (Set) (JavaScript)
傳回集合中項目的數目。  
  
## 語法  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### 參數  
 `sizeVar`  
 必要項。  任何數字。  
  
 `setObj`  
 必要項。  任意 `Set` 物件。
