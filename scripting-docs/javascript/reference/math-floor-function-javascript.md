# Math.floor 函式 (JavaScript)
傳回小於或等於其數字引數的最大整數。  
  
## 語法  
  
```  
  
Math.floor(number)   
```  
  
## 備註  
 必要的 *number* 引數是數值運算式。  
  
 傳回值為一個小於或等於其數字引數的最大整數值。
