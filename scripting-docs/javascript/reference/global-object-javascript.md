# 全域物件 (JavaScript)
為了將全域函式和常數收集到一個物件中的內建物件。  
  
## 備註  
 `Global` 物件絕不能直接使用，而且不能使用 `new` 運算子來建立。  指令碼引擎初始化時便會建立此物件，因此可以立即使用它的函式和常數。
