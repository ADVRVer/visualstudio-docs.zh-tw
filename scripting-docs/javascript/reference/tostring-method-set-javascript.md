# toString 方法 (Set) (JavaScript)
傳回集合的字串表示。  
  
## 語法  
  
```javascript  
setObj.toString()  
```  
  
#### 參數  
 `setObj`  
 必要項。  `Set` 物件。  
  
## 屬性值\/傳回值  
 集合的字串表示。
