# constructor 屬性 (Intl.NumberFormat)
指定建立格式子的函式。  
  
## 語法  
  
```javascript  
numberFormatter.constructor  
```  
  
## 備註  
 必要的 `numberFormatter` 是格式子的名稱。  
  
 `constructor` 屬性為每個具有原型之物件的原型成員。  其中包括所有內建 JavaScript 物件，但是 `Global` 和 `Math` 物件除外。  `constructor` 屬性包含建構該特殊物件之執行個體的函式參考。
