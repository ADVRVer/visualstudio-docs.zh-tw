# toString 方法 (WeakMap) (JavaScript)
傳回表示 `WeakMap` 物件的字串。  
  
## 語法  
  
```javascript  
weakmapObj.toString()  
```  
  
#### 參數  
 `weakmapObj`  
 必要項。  `WeakMap` 物件。  
  
## 屬性值\/傳回值  
 `WeakMap` 的字串表示。
