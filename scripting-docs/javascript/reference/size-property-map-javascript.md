# size 屬性 (Map) (JavaScript)
傳回對應中的項目數。  
  
## 語法  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### 參數  
 `sizeVar`  
 必要項。  任何數字。  
  
 `mapObj`  
 必要項。  任意 `Map` 物件。
