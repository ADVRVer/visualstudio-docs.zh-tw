# Math.sin 函式 (JavaScript)
傳回數字的正弦值。  
  
## 語法  
  
```  
  
Math.sin(radians)   
```  
  
## 備註  
 `radians` 引數是數值運算式，其中包含以弧度測量的角度。  
  
 傳回值為數字引數的正弦值。
