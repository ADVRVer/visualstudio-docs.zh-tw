# resolvedOptions 方法 (Intl.NumberFormat)
傳回物件，其中包含 `NumberFormat` 格式子物件的屬性和值。  
  
## 語法  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### 參數  
 `numberFormatObj`  
 要從中擷取資訊的 `NumberFormat` 物件。  
  
## 備註  
 傳回物件的屬性對應於 `NumberFormat` 物件的計算屬性。
