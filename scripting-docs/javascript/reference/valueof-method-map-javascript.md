# valueOf 方法 (Map) (JavaScript)
傳回指定物件的原始值。  
  
## 語法  
  
```javascript  
mapObj.valueOf()  
```  
  
#### 參數  
 `mapObj`  
 必要項。  `Map` 物件。  
  
## 屬性值\/傳回值  
 傳回對應執行個體。
