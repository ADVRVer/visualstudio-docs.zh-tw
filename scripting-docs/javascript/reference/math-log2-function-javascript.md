# Math.log2 函式 (JavaScript)
傳回數字以 2 為底數的對數。  
  
## 語法  
  
```  
Math.log2(x)   
```  
  
## 傳回值  
 如果 `x` 是 `NaN`，此函式會傳回 `NaN`。  如果 `x` 小於零 \(0\)，此函式會傳回 `NaN`。
