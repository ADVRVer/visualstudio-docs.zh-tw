# 乘法運算子 (*) (JavaScript)
將兩個運算式的值相乘。  
  
## 語法  
  
```  
  
result = number1*number2  
```  
  
## 參數  
 *result*  
 任何變數。  
  
 `number1`  
 任何運算式。  
  
 `number2`  
 任何運算式。
