# Math.cbrt 函式 (JavaScript)
傳回數字的立方根。  
  
## 語法  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## 備註  
 如果 `number` 是 `NaN`，結果是 `NaN`。
