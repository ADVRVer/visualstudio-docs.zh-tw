# Math.atan 函式 (JavaScript)
傳回數字的反正切值。  
  
## 語法  
  
```  
  
Math.atan(number)   
```  
  
## 備註  
 必要的 `number` 引數是需要反正切值的數值運算式。  
  
 傳回值是 `number` 引數的反正切值 \(以弧度為單位\)。
