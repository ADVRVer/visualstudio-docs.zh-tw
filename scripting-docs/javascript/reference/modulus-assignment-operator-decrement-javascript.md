# 模數指派運算子 (JavaScript)
將變數值除以運算式的值，然後將餘數指派給變數。  
  
## 語法  
  
```  
  
result %= expression  
```  
  
## 引數  
 `result`  
 任何變數。  
  
 `expression`  
 任何數值運算式。  
  
## 備註  
 使用 `%=` 運算子與以下的指定方式完全相同：  
  
```  
result = result % expression  
```
