# 除法運算子 (/) (JavaScript)
將兩個運算式的值相除。  
  
## 語法  
  
```  
  
result = number1 / number2   
```  
  
## 參數  
 `result`  
 任意數值變數。  
  
 `number1`  
 任何數值運算式。  
  
 `number2`  
 任何數值運算式。
