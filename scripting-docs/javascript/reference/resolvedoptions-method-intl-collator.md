# resolvedOptions 方法 (Intl.Collator)
傳回物件，其中包含自動分頁器物件的屬性和值。  
  
## 語法  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### 參數  
 `collatorObj`  
 要從中擷取資訊的 `Collator` 物件。  
  
## 備註  
 所傳回物件的屬性對應於 `Collator` 物件的計算屬性。
