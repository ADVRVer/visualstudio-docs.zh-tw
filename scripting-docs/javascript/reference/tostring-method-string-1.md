# toString 方法 (字串)
傳回字串的字串表示。  
  
## 語法  
  
```  
  
string.toString()  
```  
  
## 參數  
 `string`  
 必要項。  要表示為字串的陣列。  
  
## 傳回值  
 字串的字串表示。  
  
## 範例  
 以下範例說明如何搭配字串使用 **toString** 方法。  
  
```javascript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
