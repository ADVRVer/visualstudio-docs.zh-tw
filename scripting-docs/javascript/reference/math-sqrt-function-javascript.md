# Math.sqrt 函式 (JavaScript)
傳回數字的平方根。  
  
## 語法  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## 備註  
 必要的 `number` 引數是數值運算式。  
  
 如果 `number` 是負數，則傳回值為 `NaN`。
