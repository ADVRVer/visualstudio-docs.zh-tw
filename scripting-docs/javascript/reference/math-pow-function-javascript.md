# Math.pow 函式 (JavaScript)
傳回指定乘冪值的基底運算式值。  
  
## 語法  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## 參數  
 *base*  
 必要項。  運算式的基底值。  
  
 *exponent*  
 必要項。  運算式的指數值。  
  
## 範例  
 下列範例中的數值運算式會傳回 1000，也就是 base 值的 exponent 次乘冪結果。  
  
```javascript  
Math.pow(10,3);  
```
