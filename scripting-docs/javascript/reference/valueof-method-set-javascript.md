# valueOf 方法 (Set) (JavaScript)
傳回指定物件的原始值。  
  
## 語法  
  
```javascript  
setObj.valueOf()  
```  
  
#### 參數  
 `setObj`  
 必要項。  `Set` 物件。  
  
## 屬性值\/傳回值  
 傳回集合執行個體。
