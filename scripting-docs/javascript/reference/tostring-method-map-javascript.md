# toString 方法 (Map) (JavaScript)
傳回對應的字串表示。  
  
## 語法  
  
```javascript  
mapObj.toString()  
```  
  
#### 參數  
 `mapObj`  
 必要項。  `Map` 物件。  
  
## 屬性值\/傳回值  
 對應的字串表示。
