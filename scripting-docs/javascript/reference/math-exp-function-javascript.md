# Math.exp 函式 (JavaScript)
傳回 e \(自然對數的基數\) 的乘冪數。  
  
## 語法  
  
```  
  
Math.exp(number)   
```  
  
## 備註  
 必要的 `number` 引數是表示 *e* 乘冪的數值運算式。  
  
 傳回值是數字。  常數 *e* 是歐拉數字 \(Euler's number\)，大約等於 2.71828，而且 `number` 是提供的引數。
