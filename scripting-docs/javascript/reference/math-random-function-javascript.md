# Math.random 函式 (JavaScript)
傳回介於 0 到 1 之間的虛擬亂數。  
  
## 語法  
  
```  
  
Math.random( )  
  
```  
  
## 備註  
 產生的虛擬亂數介於 0 到 1 之間 \(包括 0 但不包括 1\)。也就是說，傳回的數字可以是零，但是一律小於一。  當第一次載入 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 時，就會自動啟動亂數產生器。
