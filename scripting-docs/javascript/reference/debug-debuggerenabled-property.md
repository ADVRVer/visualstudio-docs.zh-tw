# Debug.debuggerEnabled 屬性
判斷是否要針對指令碼內容啟用偵錯。  不論是否已附加偵錯工具，都可以啟用或停用偵錯。  
  
## 語法  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
