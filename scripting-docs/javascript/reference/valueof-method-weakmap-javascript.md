# valueOf 方法 (WeakMap) (JavaScript)
傳回指定物件的原始值。  
  
## 語法  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### 參數  
 `weakmapObj`  
 必要項。  `WeakMap` 物件。  
  
## 屬性值\/傳回值  
 傳回 `WeakMap` 執行個體。
