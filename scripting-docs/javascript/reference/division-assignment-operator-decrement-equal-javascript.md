# 除法指派運算子 (/=) (JavaScript)
將變數值除以運算式的值，然後將結果指派給變數。  
  
## 語法  
  
```  
  
result /= expression   
```  
  
## 參數  
 `result`  
 任意數值變數。  
  
 `expression`  
 任何數值運算式。  
  
## 備註  
 使用 `/=` 運算子與指定 `result = result / expression` 相同。
