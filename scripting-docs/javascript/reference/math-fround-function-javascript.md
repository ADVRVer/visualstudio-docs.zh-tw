# Math.fround 函式 (JavaScript)
以最接近的單精確度浮點格式傳回數字。  
  
## 語法  
  
```  
  
Math.fround(  
number  
)   
```  
  
## 備註  
 如果 `number` 是 `NaN`，結果為 `NaN`。
