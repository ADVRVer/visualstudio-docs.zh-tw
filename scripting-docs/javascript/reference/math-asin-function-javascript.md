# Math.asin 函式 (JavaScript)
傳回數值的反正弦值。  
  
## 語法  
  
```  
  
Math.asin(number)   
```  
  
## 備註  
 必要的 `number` 引數是需要反正弦值的數值運算式。  
  
 傳回值是 `number` 引數的反正弦值 \(以弧度為單位\)。
