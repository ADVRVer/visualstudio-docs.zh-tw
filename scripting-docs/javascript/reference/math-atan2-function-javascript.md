# Math.atan2 函式 (JavaScript)
傳回從 X 軸至某個點 \(*y*,*x*\) 的角度 \(以弧度表示\)。  
  
## 語法  
  
```  
  
Math.atan2(y, x)  
```  
  
## 參數  
 *x*  
 必要項。  代表直角座標 x 軸的數值運算式。  
  
 *y*  
 必要項。  代表直角座標 y 軸的數值運算式。  
  
## 備註  
 傳回值介於 *\-pi* 與 *pi* 之間。  它表示提供之 \(*y*,*x*\) 點的角度 \(以弧度表示\)。
