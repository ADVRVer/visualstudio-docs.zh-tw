# resolvedOptions 方法 (Intl.DateTimeFormat)
傳回物件，其中包含日期\/時間格式子物件的屬性和值。  
  
## 語法  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### 參數  
 `dateTimeFormatObj`  
 要從中擷取資訊的 `DateTimeFormat` 物件。  
  
## 備註  
 傳回物件的屬性對應於 `DateTimeFormat` 物件的計算屬性。
