# Math.tan 函式 (JavaScript)
傳回數值的正切函數值。  
  
## 語法  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## 備註  
 必要的 `radians` 引數是數值運算式，其中包含以弧度測量的角度。  
  
 傳回值是 `radians` 之數值引數的正切函數值。
