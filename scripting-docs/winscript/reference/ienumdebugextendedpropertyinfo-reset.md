# IEnumDebugExtendedPropertyInfo::Reset
將列舉序列重設到開頭。  
  
## 語法  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## 傳回值  
 傳回有效的 `HRESULT`，通常 `S_OK`。
