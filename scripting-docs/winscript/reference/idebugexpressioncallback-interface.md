# IDebugExpressionCallBack 介面
提供狀態事件與 `IDebugExpression` 評估的進度相關。  
  
## 方法  
 除了繼承自 `IUnknown` 的方法之外，`IDebugExpressionCallBack` 介面還會公開下列方法。  
  
|方法|描述|  
|--------|--------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|表示要評估的運算式已完成。|
