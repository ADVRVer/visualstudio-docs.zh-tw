# SCRIPT_E_PROPAGATE 錯誤碼
指令碼錯誤會傳播至呼叫端，可能是在不同的執行緒。  主應用程式應該將錯誤碼到呼叫端。  
  
## 語法  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
