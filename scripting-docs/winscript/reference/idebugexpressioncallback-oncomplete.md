# IDebugExpressionCallBack::onComplete
表示要評估的運算式已完成。  
  
## 語法  
  
```  
HRESULT onComplete();  
```  
  
#### 參數  
 這個方法不採用參數。  
  
## 傳回值  
 方法會傳回 `HRESULT`。  可能的值包括，，但不限於\)，這些在下表中。  
  
|值|描述|  
|-------|--------|  
|`S_OK`|方法成功。|  
  
## 備註  
 在運算式評估時，就會呼叫這個方法。  `IDebugExpression::GetResultAsString` 方法可能會從這個事件處理常式中。
