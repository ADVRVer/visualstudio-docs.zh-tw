# IJsDebugDataTarget::CreateStackFrameEnumerator 方法
建立堆疊框架的列舉程式。  
  
## 語法  
  
```  
HRESULT CreateStackFrameEnumerator(  
   DWORD threadId,  
   IEnumJsStackFrames **ppEnumerator  
);  
```  
  
#### 參數  
 `threadId`  
 \[in\] 目標處理序中執行的執行緒。  
  
 `ppEnumerator`  
 \[out\] 堆疊框架的列舉程式。
