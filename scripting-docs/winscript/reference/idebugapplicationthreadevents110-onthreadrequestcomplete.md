# IDebugApplicationThreadEvents110::OnThreadRequestComplete
將呼叫加入到使用 PDM 的執行緒切換執行緒中完成。  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 介面](../../winscript/reference/idebugapplicationthreadevents110-interface.md) 由 PDM v11.0 實作和更大。  位於 activdbg100.h。  
  
## 語法  
  
```cpp  
HRESULT OnThreadRequestComplete( void );  
```  
  
#### 參數  
 這個方法沒有參數。
