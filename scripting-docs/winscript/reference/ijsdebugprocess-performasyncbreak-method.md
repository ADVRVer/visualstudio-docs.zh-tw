# IJsDebugProcess::PerformAsyncBreak 方法
將指令碼引擎置於中斷模式，使其在下一個指令碼指令上中斷。  
  
## 語法  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### 參數  
 `threadId`  
 \[in\] 執行緒 ID。
