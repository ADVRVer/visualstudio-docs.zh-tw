# IActiveScriptStats::ResetStats
重設這個指令碼的統計資料。  
  
## 語法  
  
```  
HRESULT ResetStats();  
```  
  
#### 參數  
 這個方法不採用參數。  
  
## 傳回值  
 方法會傳回 `HRESULT`。  可能的值包括，，但不限於\)，這些在下表中。  
  
|值|描述|  
|-------|--------|  
|`S_OK`|方法成功。|  
  
## 備註  
 這個方法會重設這個指令碼的統計資料。
