# IDebugDocumentTextAuthor::InsertText
插入新文字輸入文件。  
  
## 語法  
  
```  
HRESULT InsertText(  
   ULONG    cCharacterPosition,  
   ULONG    cNumToInsert,  
   OLECHAR  pcharText[]  
);  
```  
  
#### 參數  
 `cCharacterPosition`  
 \[in\] 插入文字的位置。  
  
 `cNumToInsert`  
 \[in\] 要插入的字元數。  
  
 `pcharText[]`  
 \[in\] 包含字元的緩衝區插入。  
  
## 傳回值  
 方法會傳回 `HRESULT`。  可能的值包括，，但不限於\)，這些在下表中。  
  
|值|描述|  
|-------|--------|  
|`S_OK`|方法成功。|  
  
## 備註  
 這個方法插入新文字輸入文件。
