# IEnumDebugPropertyInfo::GetCount
取得 `DebugPropertyInfo` 結構數目的列舉值。  
  
## 語法  
  
```  
HRESULT GetCount (  
   ULONG* pcelt  
);  
```  
  
#### 參數  
 `pcelt`  
 \[in\] 傳回 `DebugPropertyInfo` 結構數目的列舉值。  
  
## 傳回值  
 傳回有效的 `HRESULT`，通常 `S_OK`。
