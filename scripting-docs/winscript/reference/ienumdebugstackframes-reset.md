# IEnumDebugStackFrames::Reset
將列舉序列重設到開頭。  
  
## 語法  
  
```  
HRESULT Reset();  
```  
  
#### 參數  
 這個方法不採用參數。  
  
## 傳回值  
 方法會傳回 `HRESULT`。  可能的值包括，，但不限於\)，這些在下表中。  
  
|值|描述|  
|-------|--------|  
|`S_OK`|方法成功。|  
  
## 備註  
 這個方法會將列舉型別序列重設到開頭。
