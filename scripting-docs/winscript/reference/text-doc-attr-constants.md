# TEXT_DOC_ATTR 的常數
說明文件的屬性。  
  
## 語法  
  
```  
typedef DWORD TEXT_DOC_ATTR;  
```  
  
## 常數  
  
|常數|值|描述|  
|--------|-------|--------|  
|TEXT\_DOC\_ATTR\_READONLY|0x00000001|文件是唯讀的。|  
|TEXT\_DOC\_ATTR\_TYPE\_PRIMARY|0x00000002|文件是文件樹狀結構主要檔案。|  
|TEXT\_DOC\_ATTR\_TYPE\_WORKER|0x00000004|文件是背景工作。|  
|TEXT\_DOC\_ATTR\_TYPE\_SCRIPT|0x00000008|文件是指令碼檔。|
