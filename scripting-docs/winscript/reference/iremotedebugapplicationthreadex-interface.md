# IRemoteDebugApplicationThreadEx 介面
表示執行緒在特定應用程式中。  
  
 除了繼承自 `IUnknown` 的方法之外，`IRemoteDebugApplicationThreadEx` 介面還會公開下列方法。  
  
## 依照 Vtable 順序的方法  
  
|方法|描述|  
|--------|--------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|列舉在此執行緒上所有語言的全域運算式內容。|
