# IDebugDocumentInfo 介面
在文件提供資訊，不一定會具現化。  
  
 除了繼承自 `IUnknown` 的方法之外，`IDebugDocumentInfo` 介面還會公開下列方法。  
  
## 依照 Vtable 順序的方法  
  
|方法|描述|  
|--------|--------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|傳回指定的檔案名稱。|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|傳回識別文件類型的 CLSID。|
