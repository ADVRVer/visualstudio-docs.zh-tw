# SCRIPT_E_RECORDED 錯誤碼
錯誤已傳遞指令碼引擎與主機之間。  主應用程式需要透過錯誤碼給呼叫端。  
  
## 語法  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
