# IActiveScriptSiteInterruptPoll 介面
`IActiveScriptSiteInterruptPoll` 介面允許主應用程式指定指令碼應該結束。  
  
## 方法  
 除了繼承自 `IUnknown` 的方法之外，`IActiveScriptSiteInterruptPoll` 介面還會公開下列方法。  
  
|方法|描述|  
|--------|--------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|允許主應用程式指定指令碼應該結束。|
