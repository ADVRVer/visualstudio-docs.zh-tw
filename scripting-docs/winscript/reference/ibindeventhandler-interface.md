# IBindEventHandler 介面
可讓呼叫端提供物件控制代碼所指定的事件處理常式。  
  
## 方法  
 除了繼承自 `IUnknown` 的方法之外，`IBindEventHandler` 介面還會公開下列方法。  
  
|方法|描述|  
|--------|--------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|要繫結至物件的事件。|
