# IRemoteDebugApplicationEx:RevokeBreak
取消中斷命令。  
  
## 語法  
  
```  
HRESULT RevokeBreak( );  
```  
  
#### 參數  
 無。  
  
## 傳回值  
 方法會傳回 `HRESULT`。  可能的值包括，，但不限於\)，這些在下表中。  
  
|值|描述|  
|-------|--------|  
|`S_OK`|方法成功。|
